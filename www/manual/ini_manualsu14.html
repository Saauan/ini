4.8 Imports

Since all the functions should not be defined within a single file, INI programs can start with a list of import clauses. For example:

import ”ini/examples/lib_io.ini”

Imports allow the definition and use of function libraries. In particular, it is recommended to define bindings (see Section 4.2.7) within external files and to import them when required.
