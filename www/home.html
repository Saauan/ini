Welcome to ! (INI)

INI (INI is Not ISEP) is a rule-based research language. The goal is to find out if this kind of language is suitable to write programs with high I/O and parallel execution needs.

INI is full open source and free to use.

To get started, please read the reference documentation (download PDF version). You can also checkout the online executable examples.

INI is work in progress. It is heading towards an alpha version and any help would be greatly appreciated :).
