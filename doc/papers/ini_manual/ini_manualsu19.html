6.2 Asynchronous rules

We want to be able to execute a rule asynchronously (potentially in parallel to other rules). We will add a @when() event that will take a conditional expression. For instance, in the following code, the two rules will eventually execute in parallel if a && b && c is true.

@when(a && b) { 
  // do something 
} 
@when(a && b && c) { 
  // do something 
}
